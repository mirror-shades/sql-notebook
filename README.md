# Juypter Notebook with SQL magic

This is a template for using SQL in a Jupyter Notebook

## Install
Install dependancies using the following command:   
`pip install -r requirements.txt`  

## Run
Run using the followiung command:
`juypter-notebook .`
Note that in order to run this command you will need to already have installed juypter-lab as a dependancy

## Export
To download as a PDF, jupyter-notebook uses LaTeX, which is a software for publishing. If you don't want to bother with LaTeX, you can also save as an HTML file instead. Then use can use any of the free HTML to PDF converter websites to turn your HTML page into a PDF.
